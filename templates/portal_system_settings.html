Google Workspace OAuth
@
Kosongkan = semua domain Google bisa login (asalkan email terdaftar di sistem).
Copy dan daftarkan URI ini di Google Cloud Console → Credentials → OAuth 2.0 Client.
Cara setup: Buka Google Cloud Console → Credentials → Create OAuth 2.0 Client ID (Web application) → Tambahkan Redirect URI di atas → Copy Client ID & Secret ke sini. User harus memiliki email yang sudah diisi di profil user agar bisa login via Google.
{# ── reCAPTCHA ── #}
reCAPTCHA v3 Anti-Bot Login
Lindungi halaman login dari bot & brute-force
Cara setup: Buka Google reCAPTCHA Admin → pilih Score based (v3) → daftarkan domain → copy Site Key & Secret Key ke sini. Score threshold: 0.5 (user dengan score di bawah 0.5 dianggap bot).
