Konfigurasi Telegram Bot
Dapatkan dari @BotFather di Telegram.
Digunakan jika karyawan tidak memiliki Telegram ID individual. Gunakan @userinfobot untuk mendapatkan chat ID Anda.
Cara setup Telegram Bot:
1. Chat @BotFather → /newbot → ikuti instruksi → salin token
2. Tambahkan bot ke grup, atau chat bot langsung
3. Gunakan @userinfobot untuk mendapatkan chat ID
4. Per karyawan: isi Telegram ID di halaman edit karyawan
Konfigurasi WhatsApp via OpenWA
URL server OpenWA (tanpa trailing slash).
Nama sesi di OpenWA dashboard.
Cara menghubungkan OpenWA ke aplikasi ini:
  1. Pastikan OpenWA sudah berjalan di server dan QR code sudah di-scan
  2. Cek endpoint REST tersedia: curl http://<IP-openwa>:3000/getConnectionState
  3. Isi URL OpenWA di atas, contoh: http://192.168.1.x:3000 atau http://localhost:3000 (jika satu server)
  4. Jika OpenWA dikonfigurasi dengan API key, isi kolom API Key
  5. Klik Test WA untuk verifikasi koneksi
  6. Di data karyawan, pastikan kolom Nomor HP / WA diisi dengan format 08xxxxxxxxxx — akan otomatis dikonversi ke 628xxx@c.us
Setup OpenWA di server Ubuntu (jika belum):
npm install -g @open-wa/wa-automate
wa-automate --key "api-key-anda" --port 3000 --cors
Atau via Docker: docker run -p 3000:3000 openwa/wa-automate
Penerima Notifikasi Statis

Selain email/Telegram milik karyawan, notifikasi juga dikirim ke alamat-alamat di bawah ini (misal: HRD, Manager). Pisahkan dengan koma.

Kosongkan jika tidak perlu.
Format yang diterima (pisah koma): 123456789 numeric ID personal • -100123456789 numeric ID grup • @namakanal atau namakanal username grup/channel publik. Username tanpa @ akan otomatis ditambah.
Jadwal Reminder Otomatis
Pisahkan dengan koma. Contoh: 30,14,7,1 artinya reminder dikirim 30 hari, 14 hari, 7 hari, dan 1 hari sebelum kontrak berakhir.
Reminder otomatis berjalan setiap hari pukul 08:00 saat aplikasi aktif. Gunakan tombol "Kirim Semua Reminder" di halaman Kontrak untuk trigger manual.
